Démonstration du bug role="status"

❌ Bug : Non vocalisé (élément ajouté avec contenu)

L'élément avec role="status" est créé et ajouté au DOM avec son contenu déjà présent. Le lecteur d'écran ne le vocalise pas.

✅ Solution : Vocalisé (élément déjà présent dans le HTML)

L'élément avec role="status" existe déjà dans le HTML (vide). Le JavaScript ne fait que modifier son contenu.
